A simple budget tool as an HTML file.


<h2>Features</h2> 

- Main Dashboard
- Mortgage Calculator
- Investment Calculator 
- Summary
- Export In PDF, doc and spreadsheet
- Change Dahsboard Name
- Add or remove lines
- Five themes: Light, Dark, Midnight, Warm, High Contrast
